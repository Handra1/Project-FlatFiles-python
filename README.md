# Project-FlatFiles-python
This project about the meaning of flat files in data science in python (In Indonesia)

Flat files adalah Adalah sebuah basik teks yang mengandung records dimana terdapat table data tanpa relasi didalamnya.
Header, atau bagian kepala pada file csv ini yang masing masing kolom akan dipisahkan menggunakan koma karena sesuai dengan kepanjangannya csv ini adalah Comma Separated Values. Baris paling atas dalam file adalah bagian header pada csv. Berbeda dengan file ekstensi .txt yang mana merupakan sebuah file dengan teks tidak beraturan didalamnya, bisa saja terdapat koma, tanda dollar, banyak spasi dan lain-lain.
